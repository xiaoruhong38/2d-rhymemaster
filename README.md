# Rap写词小游戏 MVP（半句填空）

一个基于 Web 的 Rap 写词小游戏：每一关给出一句不完整的上半句，玩家需要在 8×8 汉字棋盘中找出**押韵**的下半句（4–10 字）。

## 游戏规则（半句填空）

1. **游戏界面**：8×8 的汉字网格，共 64 个汉字
2. **每关提示**：上方显示一句不完整 Rap 歌词（上半句），例如：`穿过幽暗的岁月，___________`
3. **操作方式**：
   - 在棋盘中**连续点选相邻汉字（上下左右）**，组成一条 4–10 字的下半句
   - 棋盘上会**橙色高亮**所有符合当前韵脚的汉字，便于快速取材
   - 点击「检测下半句」进行判定
4. **判定机制**：
   - **有效性**：优先检查是否命中本关的 `acceptableLowers`（答案库），其次检查是否命中全局 `validSentences`，最后做简单通顺度兜底
   - **押韵**：使用 `rhymeGroups` 做单押判断——比较**上半句的韵脚字**与**下半句最后一个字**的韵母组是否一致
5. **过关条件**：下半句既“有效”又“押韵”，则自动进入下一关；下一关会重新随机一种常见韵脚并给出上半句

## 技术栈

- HTML5
- CSS3（使用Grid布局和动画效果）
- 原生JavaScript（无依赖）

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 按上方提示在棋盘中点选连续相邻汉字组成下半句
3. 点击「检测下半句」，通过后进入下一关

## 游戏特性

- ✅ 8×8汉字网格布局
- ✅ 半句填空关卡（上半句提示）
- ✅ 连续路径选句（4–10字）
- ✅ 下半句有效性判定（关卡答案库 + 全局歌词库 + 简易兜底）
- ✅ 单押押韵检测（基于韵母匹配）
- ✅ 过关自动进入下一关
- ✅ 重新开始功能
